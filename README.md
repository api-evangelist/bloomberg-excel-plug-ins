# Bloomberg Excel Plug-ins

Bloomberg Excel Plug-ins integrate Bloomberg market data, analytics, and functions directly into Microsoft Excel. The Bloomberg Add-in for Excel provides formula functions (BDH, BDP, BDS, BQL) to pull real-time and historical data, enabling quantitative analysis, financial modeling, and reporting without leaving the spreadsheet environment.

## APIs

- **Bloomberg Excel Add-in** - BDP, BDH, BDS formula functions for Excel
- **Bloomberg BQL in Excel** - Bloomberg Query Language integration in Excel
